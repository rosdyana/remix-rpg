## How to setup?
1. Create a Python virtual environment 

```
python -m venv .venv
```

2. Activate the virtual environment

```
.\.venv\Scripts\activate
```

3. Install the requirements

```
pip install -r requirements
```

4. Run the game

```
cd .\code\;python main.py
```
